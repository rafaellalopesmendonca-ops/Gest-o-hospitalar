# Gestao-hospitalar
Sistema de gestão hospitalar focado em triagem inteligente e recepção de pacientes. Desenvolvido com JavaScript vanila, implementa lógica de prioridade (Protocolo de Manchester), filtros de busca em tempo real e persistência de dados local.

# 🏥 Gestão Hospitalar Inteligente

Projeto desenvolvido para otimizar o fluxo de atendimento em recepções hospitalares, unindo minha experiência na área administrativa com o desenvolvimento de software.

## 🚀 Funcionalidades
- **Triagem Digital:** Cadastro com classificação de risco por cores (Emergência, Urgente e Eletivo).
- **Dashboard Dinâmico:** Contadores automáticos de pacientes totais e em espera.
- **Busca Inteligente:** Filtro em tempo real por nome ou CPF.
- **Relatórios:** Função de impressão formatada para relatórios de turno.
- **UX Hospitalar:** Interface clean, responsiva e com máscaras de dados para evitar erros de digitação.

## 🛠️ Tecnologias
- HTML5 / CSS3 Moderno
- JavaScript (ES6+)
- Font Awesome (Ícones)
- LocalStorage (Persistência de dados)
